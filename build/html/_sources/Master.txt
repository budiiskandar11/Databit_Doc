Master Bahasa
-------------

Setting Bahasa hanya digunakan untuk format template tanggal dan angka. Untuk bahasa, kami sarankan tetap menggunakan 
bahasa inggris karena lebih general.

Untuk melakukan setting Bahasa, dapat diakses melalui :

::

   Menu > Settings > Translation > Language > pilih English
   
maka akan muncul tampilan seperti ini :

.. image::  image/language.png

::

   Lakukan setting language seperti gambar diatas.
